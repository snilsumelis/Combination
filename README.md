# Java Program to Calculate the Combination (nCr)

## Overview

This Java program calculates the combination (nCr) of two numbers, n and r, provided by the user. It uses the formula C(n,r) = n! / (r! * (n-r)!).

## Usage

1. **Clone the repository:**
   - `git clone https://github.com/your-username/combination-calculator.git`

2. **Navigate to the project directory:**
   - `cd combination-calculator`

3. **Compile and run the Java program:**
   - `javac Main.java`
   - `java Main`

4. **Enter numbers when prompted:**
   - `Bir sayı giriniz:` (Enter the value for n)
   - `Bir sayı giriniz:` (Enter the value for r)

5. **View the result:**
   - The program will calculate and display the combination (nCr) based on the provided values of n and r.
